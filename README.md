## Deployment Options

### Option 1: GitHub Pages (Free)

1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select the main branch as source
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify (Free & Recommended)

1. Go to [Netlify](https://www.netlify.com/)
2. Sign up/login
3. Drag and drop your project folder OR
4. Connect your GitHub repository
5. Your site will be live instantly with a free `.netlify.app` domain
6. You can add a custom domain later

### Option 3: Vercel (Free)

1. Go to [Vercel](https://vercel.com/)
2. Sign up/login
3. Import your project
4. Deploy with one click

### Option 4: Any Static Hosting

This is a static website, so it can be hosted on:
- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting
- AWS S3 + CloudFront
- Any web hosting service

## Setup Instructions

1. **Add Your Profile Photo**
   - Place your image file in the root directory
   - Name it `profile-photo.jpg`
   - Or update the image path in `index.html` (lines 46 and 62)

2. **Customize Content**
   - Edit `index.html` to update your information
   - Modify `styles.css` to change colors/themes
   - Update `script.js` for additional functionality

3. **Test Locally**
   - Open `index.html` in a web browser
   - Or use a local server:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js (if you have http-server installed)
     npx http-server
     ```

4. **Deploy**
   - Choose one of the deployment options above
   - Follow the specific instructions for your chosen platform
